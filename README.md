# 🍔 Cardápio Online

Projeto de **cardápio online** desenvolvido com foco em prática de **front-end**, simulando o funcionamento de um sistema de pedidos para uma lanchonete.

O sistema permite visualizar produtos, adicionar itens ao carrinho, remover itens e finalizar o pedido enviando a mensagem diretamente para o WhatsApp.

---

## 🚀 Funcionalidades

* Visualização de produtos do cardápio
* Adição de itens ao carrinho
* Remoção de itens do carrinho
* Cálculo automático do total do pedido
* Validação de endereço para entrega
* Verificação de horário de funcionamento do restaurante
* Envio do pedido diretamente para o WhatsApp

---

## 🛠 Tecnologias utilizadas

* **HTML5**
* **CSS3**
* **JavaScript**
* **TailwindCSS**
* **Toastify**

## 📚 Sobre o projeto

Este projeto foi desenvolvido **para fins de estudo e prática de desenvolvimento front-end**.

O projeto foi construído acompanhando um tutorial do YouTube do canal **Sujeito Programador**.

Durante o desenvolvimento, foram feitas pequenas adaptações e commits organizados para praticar:

* Estruturação de código
* Manipulação do DOM
* Lógica de carrinho de compras
* Uso de commits semânticos
